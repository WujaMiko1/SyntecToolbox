Ladder Viewer
v4.0 • Wizualny analizator PLC

Przegladarka skompilowanych programów PLC (.lad) dla sterowników Syntec CNC. Pełne tłumaczenie chińskich etykiet na polski, cross-reference rejestrów, mapa zależności.

  • Dekodowanie binarnych .lad z etykietami Big5
  • 229+ tłumaczeń chiński → polski
  • 12 zakładek: Mapa, Xref, Diagram, I/O, System, Pomoc...
  • Import IoMap XML — mapowanie sprzętowe I/O
  • Globalne wyszukiwanie (Ctrl+K), notatki, eksport JSON
  • Porównanie wielu plików .lad
